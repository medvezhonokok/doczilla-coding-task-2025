# Инструкция по запуску 

## 1. Проверьте версию Java

Для корректной работы программы требуется **Java версии 21 или выше**.

```bash
java --version
```

Если Java не установлена или версия ниже 21, скачайте и установите:

- **[Oracle JDK](https://www.oracle.com/java/technologies/downloads/)** (рекомендуется)
- **[OpenJDK](https://openjdk.org/)**

## 2. Подготовка файлов

Убедитесь, что у вас есть следующая структура файлов:

```
src/java/
├── exceptions/
│   └── FFException.java
├── model/
│   ├── Color.java
│   ├── Drop.java
│   ├── Flask.java
│   ├── Game.java
│   └── Move.java
└── Solution.java
```

## 3. Компиляция программы

Перейдите в директорию с исходным кодом и выполните компиляцию:

```bash
# Переход в директорию с Solution.java
cd src/java

# Компиляция всех Java-файлов
javac *.java
```

Если компиляция завершилась успешно, в директории появятся файлы `.class`.

## 4. Запуск программы

Запустите программу командой:

```bash
java Solution
```

Программа выполнит тест по умолчанию:

```java
new Object[][]{
        {1,2,1},
        {2,1,2},
        {},
        {}
        },3
```

## 6. Формат входных данных

Конструктор класса `Game` принимает два параметра:

1. **Двумерный массив `Object[][]`** размером N×V, где:
    - `N` — количество пробирок
    - `V` — ёмкость каждой пробирки
    - Элементы могут быть числами, строками или символами
    - Пустые пробирки обозначаются пустыми массивами `{}`

2. **Целое число** — ёмкость пробирки (максимальное количество капель)

Пример:

```java
// 4 пробирки, ёмкостью 3
new Game(
    new Object[][] {
    {
        1, 2, 1
    },  // Пробирка 0
    {
        2, 1, 2
    },  // Пробирка 1
    {
    },         // Пробирка 2 (пустая)
    {
    }          // Пробирка 3 (пустая)
},
        3               // Ёмкость каждой пробирки
        )
```
